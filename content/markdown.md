* `>` : 引用，可多级

* `*+-`表示无序列表，`1.`表示有序列表
* # 表示标题
* 四个空格或一个tab表示代码

    This is code

* `*text*`表示强调
* ======定义h2，------定义h1
* href: `[name](http://www.txt "title")`
* image: `![name](http://www.txt "title")`
* 引用 [wiki][1]
